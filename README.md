## Task 3 SQL 
This task focuses on analyzing an Ecommerce dataset using SQL in SQLite.
The dataset contains transaction details such as InvoiceNo, Description, Quantity, UnitPrice, CustomerID, and Country.

Queries Included:
- Basic data extraction using SELECT
- Filtering records using WHERE
- Sorting results using ORDER BY
- Sales and revenue analysis using GROUP BY with SUM, AVG, COUNT
- Filtering grouped results using HAVING
- Advanced analysis using subqueries
- Creation of views for customer revenue summaries
- Query optimization using indexes
- Handling missing values using IFNULL / NULL checks

Deliverables Submitted:
- SQL queries file (SQLQueries.sql)
- Screenshots of outputs for each query execution
